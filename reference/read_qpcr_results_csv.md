# Read qPCR 'Results' or 'Standard Curve Result' table from a CSV file

Read qPCR 'Results' or 'Standard Curve Result' table from a CSV file

## Usage

``` r
read_qpcr_results_csv(file)
```

## Arguments

- file:

  path to file; see
  [`readr::read_csv()`](https://readr.tidyverse.org/reference/read_delim.html)
